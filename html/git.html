Git

  • git add - добавление файлов в индекс.
  • git status - просмотр списка изменённых файлов.
  • git commit - сохранение изменений.
  • git push - выгрузка изменений в удаленный репозиторий.
  • git log - просмотр истории коммитов в репозитории.
  • git pull - получение изменений из удалённого репозитория в локальный.
  • git clon - клонирование репозитория из удалённого хранилища на локальную машину.
    • git branch название ветки - создание ветки. git checkout название ветки - переход на ветку.
  • git init - создание нового репозитория в пустой директории или превращение существующей директории в новый репозиторий.
